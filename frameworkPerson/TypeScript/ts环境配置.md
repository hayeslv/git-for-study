#### ts-node

Node环境的typescript解释执行器。REPL（Read eval print loop）

```shell
npm i ts-node -g
```

可以用ts-node执行文件

```shell
ts-node some.ts
```



#### 配置文件：tsconfig.json

```json
{
  "compilerOptions": {
    // 书写你的配置
  }
}
```



#### tsc（typescript compiler）

一个ts的编译器

```shell
npm i tsc -g
```
